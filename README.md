# InnovationB_Qt_HearthStone

#### 介绍
哈尔滨工业大学（深圳）大一创新训练课B（可简单理解为软件工程降阶版）大作业，自选题目“简易炉石传说”，借此机会练习Qt图形界面开发基础。基本实现了卡组创建与编辑、嘲讽/圣盾/冲锋/风怒/潜行等卡牌效果与局域网对战功能。使用的语言是C++。软件安装包在release中，其中有两套，详见release说明。
